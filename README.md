# College-Announcement-Management-System
A responsive college announcement dashboard with admin login, year-wise notice filtering, and AJAX-based CRUD operations.

The announcement dashboard is built using **PHP, MySQL, HTML, CSS, JavaScript, and jQuery**.

This project allows an admin to log in and manage college announcements year-wise for different student batches.

---

## Features

- Admin login authentication using PHP sessions
- Add announcements with:
  - Title
  - Description
  - Category
  - Year
- Year-wise announcement dashboard:
  - 1st Year
  - 2nd Year
  - 3rd Year
  - 4th Year
- Common announcements using **All Years**
- AJAX-based loading without page refresh
- Delete announcements
- Category-based color cards
- Responsive modern UI
- Animated active year tabs
- Secure session-based access

## How to Run

1. Install XAMPP
2. Start Apache and MySQL
3. Move project folder to:

```text
C:\xampp\htdocs\
```

4. Open phpMyAdmin
5. Create database:

```sql
announcement_db
```

6. Import required tables
7. Open in browser:

```text
http://localhost/project-folder-name/login.php
```
